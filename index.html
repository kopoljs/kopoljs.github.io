What is kopol.js?

Kopol is a mini javascript visualization recommender. Kopol recommends you the ranked list of the most effective visualizations (Table, Line Chart, Bar Chart, Scatterplot, and Pie Chart) in response to the given tasks and data attributes. The underlying data used for training kopol.js is based on thousands of data points called from a crowdsourced experiment tested the effectiveness of five visualization types across ten common data analysis tasks using two real-world datasets. Kopol ranks visualizations based on the combination of users' performance time, accuracy, and preferences.

Getting Started

To use kopol.js in browser, include kopol.js file.

<script src="/js/kopol.min.js"></script>

getChartPeformance([task, attr1, attr2])

Returns a ranked list of 5 visualizations based on their effectiveness in response to the given task and data attribute types. Data attribute types supported by the current version include ordinal, numerical, and nominal. 

getChartPeformance([task, attr1, attr2], function(val){
     // do something
});

Example: 

 //Code 
getChartPeformance( ["Retrieve","Ordinal","Nominal"], function(val)
{
  console.log("best to worse: ",val);
});

 //Output
best to worse:  [ 'Table', 'BarChart', 'PieChart', 'Scatterplot', 'LineChart' ]
