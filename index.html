What is Kopol.js?

Kopol is a light javascript visualization recommender. Kopol recommends you the ranked list of the most effective visualizations (Table, Line Chart, Bar Chart, Scatterplot, and Pie Chart) in response to the given tasks. The underlying data used for training kopol.js is based on results from a crowdsourced experiment tested the effectiveness of five visualization types across ten common data analysis tasks using two real world datasets. Kopol ranks visualizations based on combination of users' performance time, accuracy, and preferences.

Getting Started!

For Node.js, first install module from npm. 

 npm install kopol

Then get the module 

 var kopol = require('kopol');

getChartPeformance([task, attr1, attr2])

Returns a ranked list of 5 visualizations based on their effectiveness in response to the given task and data attribute types. 

kopol.getChartPeformance([task, attr1, attr2], function(val){
     // do something
});

Example: 

 //Code 
kopol.getChartPeformance( ["Retrieve","Ordinal","Nominal"], function(val)
{
  console.log("best to worse: ",val);
});

 //Output
best to worse:  [ 'Table', 'BarChart', 'PieChart', 'Scatterplot', 'LineChart' ]
